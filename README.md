# Fitness App

A modern fitness app that suggests home and gym exercises based on user weight and height.

## Features
- User inputs weight and height
- Choose between home or gym exercises
- Personalized exercise suggestions
- Modern, responsive UI

## Tech Stack
- Frontend: React (client)
- Backend: Node.js + Express (server)

## Getting Started

### Client
```
cd client
npm install
npm start
```

### Server
```
cd server
npm install
npm start
```
